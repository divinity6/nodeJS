## Validation

- 검증 이해


- 사용자 입력 유효성 검사

---

### Why Validate?

- 유효성 검사가 왜 중요한가?


- 해당 유효성 검사를 어떻게 구현할 수 있는가?


- 즉, 처리할 데이터가 원하는 형식의 데이터인지 검사하는 것


- validation 체크가 없다면, 유효하지 않은 이메일로 로그인을 시도 하더라도 허용된다
  - ( 사용자가 잘못된 정보를 입력하는 것을 막지 않고 있다 )
  - 이럴경우 Controller/Middleware 즉, 데이터베이스에 저장하기 직전에 유효성 검사단계를 추가한다
